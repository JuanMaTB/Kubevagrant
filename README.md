# Kubevagrant
This is a compilation resources for Ubuntu that is deploying:

   * Vagrant
   * Minikube
   * Docker
   * Kubectl
   * Ansible 
   * Helm Charts
 
 The goal is to have a fully operational machine to work with docker and kubernetes


# Requirements
    * Virtualbox
    * Vagrant
    
    
# Orchestration

  1- Open your terminal on this repository
  2- type: Vagrant up
  3- type: Vagrant ssh
